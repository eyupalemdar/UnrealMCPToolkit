# UE Component Recipes

Component recipes are short, verified notes for Unreal/UMG/CommonUI widgets that
agents commonly misuse.

Before experimenting with a component, read its recipe. If no recipe exists,
follow `../UE_COMPONENT_PROTOCOL.md`, create a minimal probe, then add a recipe.

## Initial Recipes

- `ScrollBox.recipe.md`
- `ScaleBox_SizeBox.recipe.md`
- `WidgetSwitcher.recipe.md`

Use `TEMPLATE.recipe.md` when adding a new component recipe.
